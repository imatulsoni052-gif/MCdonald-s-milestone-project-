# 🍔 McDonald's Sales Performance Analysis Dashboard

## 📌 Project Description

This project focuses on analyzing McDonald's sales data using an interactive Excel dashboard. The goal is to uncover key business insights related to sales performance, customer behavior, and product popularity to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing menu items
* Understand customer ordering patterns
* Evaluate time-based sales trends
* Track category-wise revenue contribution

---

## 📊 Dashboard Overview

The dashboard provides a comprehensive view of sales data with interactive features for better analysis.

### 🔹 Key Features

* Interactive slicers (Date, Category, Menu Item)
* Dynamic KPIs for quick insights
* Category-wise and item-wise revenue breakdown
* Time-based sales trends (daily, monthly, weekends vs weekdays)
* Top-performing products comparison

---

## 📈 Key Metrics

* **Total Sales:** $61,716.91
* **Total Orders:** 5,370
* **Average Items per Order:** 2.3

---

## 🧹 Data Cleaning Process

* Removed duplicate records
* Handled missing/null values using forward & backward fill
* Standardized data formats for consistency

---

## 🔍 Analysis & Insights

### 🍔 Sales Performance

* Burgers and combo meals generate the highest revenue
* A small number of items contribute to the majority of total sales

### 📦 Order & Popularity Trends

* Weekend orders are significantly higher than weekdays
* Signature burgers are the most frequently ordered items

### ⏰ Time-Based Trends

* Peak sales occur during lunch and dinner hours
* Order volume varies significantly throughout the day

### 📅 Monthly & Category Trends

* Seasonal patterns observed in sales
* Beverage sales increase during summer months

### 🏆 Top Products

* Top 5 menu items drive a large portion of revenue
* Product bundling can increase average order value
